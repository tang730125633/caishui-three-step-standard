# 00-从零环境安装（客户机器只有 OpenClaw）

> 适用场景：客户电脑上没有 Python、没有依赖环境，只有 OpenClaw 和 AI 终端。

---

## 一、最低要求

- 操作系统：Windows 10/11 或 macOS
- 已安装：OpenClaw
- 可访问网络（用于下载 Python / pip 包）

---

## 二、先做环境自检（AI 可直接执行）

在终端运行：

```bash
python3 --version || python --version
pip3 --version || pip --version
```

如果显示 `command not found`，说明 Python 未安装，按下面步骤安装。

---

## 三、Python 安装（官方推荐）

## 3.1 Windows

### 方式 A（推荐）：Microsoft Store
1. 打开 Microsoft Store
2. 搜索 `Python 3.12`（或最新版）
3. 安装后重新打开终端

### 方式 B（官网安装包）
- 官方下载页：<https://www.python.org/downloads/windows/>
- 建议版本：Python 3.11+ 或 3.12+
- 安装时务必勾选：`Add python.exe to PATH`

### 方式 C（命令行）
```powershell
winget install -e --id Python.Python.3.12
```

---

## 3.2 macOS

### 方式 A（推荐）：Homebrew
```bash
brew install python
```

### 方式 B（官网安装包）
- 官方下载页：<https://www.python.org/downloads/macos/>

---

## 四、安装项目依赖

在仓库根目录执行：

```bash
python3 -m pip install --upgrade pip
python3 -m pip install requests pandas openpyxl playwright websocket-client
python3 -m playwright install chromium
```

> 若是 Windows 无 `python3`，可把 `python3` 改成 `python`。

---

## 五、浏览器准备（必须）

## 5.1 启动 Edge 调试模式（端口 9223）

### Windows PowerShell
```powershell
taskkill /F /IM msedge.exe
& "C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe" --remote-debugging-port=9223 --no-first-run --no-default-browser-check
```

### macOS
```bash
pkill -f "Microsoft Edge" || true
/Applications/Microsoft\ Edge.app/Contents/MacOS/Microsoft\ Edge --remote-debugging-port=9223 --no-first-run --no-default-browser-check
```

## 5.2 登录财税通
- 打开：`https://cst.uf-tree.com`
- 登录企业账号

## 5.3 验证调试端口
打开：
- `http://127.0.0.1:9223/json/version`
- `http://127.0.0.1:9223/json/list`

返回 JSON 即正常。

## 七、关于“把 Python 安装包放仓库”的建议

可以做，但默认不建议直接把安装包放 GitHub 仓库根目录，原因：
- 安装包体积大（仓库膨胀）
- 版本更新快，维护成本高
- 不同系统/架构需多份包

更推荐：
1. 文档里提供官方下载链接（当前方案）
2. 如需离线交付，单独做 `offline-installers/` 发布页或网盘包（不进主仓）
3. 在 README 中增加“离线交付说明”入口

## 八、常见问题

### Q1: pip 安装超时
可先升级 pip，或更换网络环境后重试。

### Q2: Playwright 报浏览器未安装
执行：
```bash
python3 -m playwright install chromium
```

### Q3: 找不到已登录页面
通常是 Edge 没用 9223 启动，或未在该窗口登录财税通。
